# عداد رحلة السيارة PWA

تطبيق ويب تقدمي للآيفون والأندرويد يتتبع رحلة السيارة عبر GPS، مع واجهة عربية RTL.

## الميزات

- سرعة حالية بالكيلومتر/ساعة.
- أعلى سرعة للرحلة الحالية.
- متوسط سرعة الرحلة ومتوسط الحركة.
- مسافة الرحلة ومدة الرحلة.
- بطاقة آخر رحلة في الشاشة الرئيسية.
- سجل رحلات محفوظ محلياً على الجهاز.
- إضافة رحلة يدوية عند نسيان تشغيل GPS.
- تصدير واستيراد نسخة احتياطية JSON.
- ميزة إبقاء الشاشة مفعلة أثناء الرحلة عبر Wake Lock عند دعم المتصفح.
- يعمل كـ PWA ويمكن إضافته إلى الشاشة الرئيسية.

## ملاحظات مهمة

- يجب تشغيل التطبيق عبر HTTPS حتى تعمل خدمة الموقع وميزات PWA بشكل صحيح.
- على iPhone: افتح الرابط في Safari ثم مشاركة ← إضافة إلى الشاشة الرئيسية.
- Wake Lock يعتمد على دعم المتصفح والنظام، لذلك قد يختلف بين الأجهزة.
- لا تستخدم الهاتف أثناء القيادة. الأفضل تثبيته على حامل أو استخدامه من قبل الراكب.

## الملفات

- `index.html`: التطبيق الكامل.
- `manifest.json`: إعدادات PWA.
- `service-worker.js`: التخزين المؤقت والعمل دون اتصال بعد أول تحميل.
- `icon-192.png` و `icon-512.png`: أيقونات التطبيق.
